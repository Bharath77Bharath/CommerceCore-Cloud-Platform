🛒 CommerceCore Cloud Platform

Spring Boot • PostgreSQL • Docker • AWS EC2 • GitHub Actions

About

A production-style Ecommerce REST API built using Spring Boot and PostgreSQL, containerized with Docker, deployed on AWS EC2, and automated using GitHub Actions CI/CD.

Features

  • Product Management (CRUD)
  • Search Products
  • Pagination Support
  • Order Management
  • Review Management
  • Product Image Management
  • Dockerized Deployment
  • AWS EC2 Hosting
  • GitHub Actions CI/CD

API Endpoints

GET /api/products

POST /api/products

PUT /api/products/{id}

DELETE /api/products/{id}

POST /api/orders

GET /api/orders/{id}

POST /api/reviews

POST /api/products/{id}/images
